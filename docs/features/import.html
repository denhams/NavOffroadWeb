Overview

You can add map features from many different sources. These can be waypoints, tracks, routes, land use areas, etc., and even complete maps!

Map feature formats

You can import features from GeoJSON, GPX, KML, TileJSON, Cloud Optimizied GeoTIFF, and map tile server sources. Imported features are stored locally on your device.

Pins

Drop pins on the map to record points of interest. You can select your own color for the pin and an icon from our extensive library. Share these with your friends!

Map feature types

Your imported features can be symbols (for waypoints, points of interest, etc.), lines (for tracks, routes, etc.) and areas (for land use, etc.). Choose what you want to import from a source. Map tile sources overlay a complete third-party map, e.g. OpenStreetMap, USA Topo map.
