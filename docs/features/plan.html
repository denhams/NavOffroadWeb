Overview

The NavOffroad Planner web app is for creating, viewing, and sharing maps. It uses GeoJSON, an open source spatial data format, but also accepts KML, GPX, CSV, TopoJSON and TileJSON. You can use it to create and manage map data (routes, points of interest, etc.) for the NavOffroad iPad® and iPhone® app.

Create and edit map data

Import your existing map data or create new features using the NavOffroad Planner.

Build routes

Build your own routes by connecting locations on the map. Use the NavOffroad iPad® and iPhone® app to get turn-by-turn navigation guidance along the route.

Get map data

We have information about many map data sources that you can use.
