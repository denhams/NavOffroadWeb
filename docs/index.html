NavOffroad is an iOS app (iPhone, iPad, and Macs with Apple Silicon) that provides off-road maps, trails, routing with turn-by-turn navigation, 3D terrain, sat imagery, weather, and CarPlay®. You can import your own map data.

Use the companion web app to plan your trip and create and edit your own map data.
